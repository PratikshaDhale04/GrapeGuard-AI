# 📌 Project Overview

Downy Mildew is one of the most harmful diseases affecting grape cultivation. It is caused by *Plasmopara viticola* and spreads rapidly in warm, humid, and rainy environmental conditions. If not detected early, it can lead to severe crop damage, poor fruit quality, and major financial losses for farmers.

This project presents a **Machine Learning-based Smart Prediction System** that predicts the risk of Downy Mildew disease in grape crops using environmental and crop-related data such as:

- 🌧️ Rainfall
- 💧 Humidity
- 🌡️ Temperature
- 🌿 Crop Growth Stage
- 📊 Historical Disease Data

The system helps farmers take preventive measures before the disease spreads widely, supporting smart and sustainable agriculture.

# 🤖 Machine Learning Algorithms Used

## 1️⃣ Support Vector Machine (SVM)
SVM is used for classification and pattern detection. It separates disease and non-disease conditions using a hyperplane
## 2️⃣ Random Forest
Random Forest uses multiple decision trees and combines their outputs for accurate predictions.
## 3️⃣ Decision Tree
Decision Tree predicts disease risk using simple rule-based conditions.

# ⚙️ Working Process

## Step 1: Data Collection
Environmental and crop-related data is collected from:
- Weather conditions
- Vineyard observations
- Historical disease records

---

## Step 2: Data Preprocessing
Collected data is cleaned and prepared by:
- Removing missing values
- Formatting data
- Feature selection

---

## Step 3: Model Training
Machine Learning algorithms are trained using historical datasets.

The model learns patterns like:
- High humidity + rainfall → High disease risk

---

## Step 4: Model Testing and Evaluation
The trained model is tested using unseen data to check performance.

Evaluation metrics used:
- Accuracy
- Precision
- Recall
- F1-Score

---

## Step 5: Prediction
New environmental data is given to the model.

The system predicts:
- Low Risk
- Medium Risk
- High Risk
