# 🍇 Smart Prediction of Downy Mildew in Grapes using Machine Learning

## 📌 Project Overview
This project predicts **downy mildew disease in grapes** using Machine Learning techniques. It analyzes environmental factors like temperature, humidity, and rainfall to estimate the risk level of disease occurrence. The goal is to help farmers take timely preventive actions and reduce crop loss.

## 🚀 Features
- Predicts downy mildew risk in grapes
- Uses environmental parameters (temperature, humidity, rainfall)
- Machine Learning model (SVM / Decision Tree)
- Simple input and prediction system
- Helps in smart and sustainable agriculture

## 🧠 Technologies Used
- Python
- Machine Learning (Scikit-learn)
- Pandas, NumPy
- Matplotlib (for visualization, if used)

## 📊 Working Process
1. Data collection of environmental conditions  
2. Data preprocessing and cleaning  
3. Model training using ML algorithms  
4. Testing and evaluation  
5. Prediction of disease risk based on input data  

## 🌾 Objective
To assist farmers in early detection of downy mildew disease and reduce crop loss by enabling data-driven decision-making.

## 📈 Outcome
The system provides accurate predictions of disease risk, helping in better crop management and reducing unnecessary pesticide usage.
